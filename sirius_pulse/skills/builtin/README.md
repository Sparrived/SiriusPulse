# 内置 Skills

本目录保存随框架发布的 Skill。它们会被 Skill 注册器扫描，并可在 WebUI 中按人格启停。

当前文件包括：`bash.py`、`container_admin.py`、`interaction.py`、`file_upload.py`、`group_management.py`、`chat_with_developer.py`、`desktop_screenshot.py`、`github_monitor.py`、`learn_term.py`、`micro_device_status.py`、`reminder.py`、`web_lookup.py`、`qq_member_info.py`。

新增、删除或重命名 Skill 后同步 `docs/extensions/skill-builtin.md`。高风险能力必须设置权限限制。
